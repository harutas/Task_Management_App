# task_management_app
Recursionの課題として作成したタスク管理アプリです。  
1. Section名を入力し、「ADD SECTION」ボタンで新しいセクションを作成します。  
2. 各セクションの「+」ボタンゴミ箱ボタンでタスクの追加、削除ができます。  
3. 各タスクのテキストの編集、完了、お気に入り登録、削除ができます。
